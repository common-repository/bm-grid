BM Grid WordPress plugin will make your portfolio and gallery easily. The unique Drag&Drop custom portfolio builder is built-in.

== Description ==

BM Grid is a WordPress grid plugin which allows you create unlimited layout. There is the unique Drag&Drop custom portfolio builder can help you create any kinds of image list or blog. 

[vimeo https://vimeo.com/205508885]

[BM Grid WordPress Plugin Live Demo](http://themes.uiueux.com/bm-grid-wordpress-plugin/)

#### Features

* Shortcode supported, insert everywhere you want
* Fully Responsive 
* Layout template: Custom Grid(Metro) / Masonry / Standard Grid / Irregular
* Item Spacing: 0/10/20/30/40/
* Columns: 2/3/4/5/6 (for Custom Grid, there are 1-12)
* Filter: left/center/right/no filter
* What Thumbnail Does: Open the post item, Open lightbox and Open external link
* Pagination: Load more button and Infiniti scroll (not for Custom Rid)

== Installation ==

1. In your WordPress admin panel, go to Plugins > New Plugin, search for <strong>BM Grid</strong> and click "Install now”
2. Alternatively, download the plugin installable file “bm-grid.zip”, un-zip it, upload the “bm-grid” folder to your plugins directory, which usually is /wp-content/plugins/.
3. Activate the plugin


== Frequently Asked Questions ==

#### Why can’t see thumbnail?
The featured image should be set on each post in grid list.
